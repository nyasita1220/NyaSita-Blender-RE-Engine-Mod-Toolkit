# NyaSita RE Engine ToolKit (NST Toolkit)

Blender 4.5 插件。为 RE 引擎游戏（Resident Evil 系列、Devil May Cry 5 等）制作 MOD 的工具箱。

## 安装

1. 把 `NyaSitaToolKit` 文件夹打包为 `.zip`
2. Blender → Edit → Preferences → Add-ons → Install → 选择 zip → 启用
3. 3D View 侧边栏 (N 键) → **NST Toolkit** 标签页

## 模式说明

| 标签 | 含义 |
|---|---|
| 🟢 Object | 物体模式 |
| 🟡 Edit | 编辑模式（骨骼编辑模式） |
| 🔵 Pose | 姿态模式 |

---

## 功能

### 合并到活跃物体 `🟢 Object`

| 按钮 | 说明 |
|---|---|
| **根据名称模板合并** | 选中 N 个物体 → 合并到活跃物体，按模板命名。`@n` = 第一个选中物体的名字 |
| **以原物体名称合并** | 合并后直接叫第一个选中物体的名字 |

模板示例：`LOD_1_Group_0_Sub_1__@n_Mat` → 合并后为 `LOD_1_Group_0_Sub_1__Cube_Mat`

支持实时预览。

---

### LOD 改名 & 随机命名为游戏网格名称 `🟢 Object`

**LOD/Group/Sub 数字批量修改**：选中物体 → 改数字 → 应用到选中。支持 Strip .NNN 和自定义后缀。

**随机命名**：从名称池 (JSON 文件，跟插件走) 随机抽取名字给选中物体命名。

| 按钮 | 说明 |
|---|---|
| `[▼]` | 切换名称池 |
| `[+]` | 新建空白名称池 |
| `[📝]` | 在 Blender 文本编辑器中打开名称池 |
| `[✓]` | 保存文本编辑器中的修改 |
| `[✕]` | 删除名称池 |
| `[❗]` | 检测名称池中是否有重复名字 |

**格式选项**：
- **Noesis** → `LOD_X_Group_Y_Sub_Z__Name_Mat`
- **RE Mesh** → `Group_Y_Sub_Z__Name_Mat`

**使用面板数字**：勾选 → 用 LOD 改名面板的数字；不勾选 → 保留每个物体原有的数字。

**同名物体分组**：LOD/Group/Sub 数字不同但身体名相同的物体会分到同一组，用同一个池名字，数字各自保留。

---

### 防脊柱移位工具 `🟡 Edit / 🔵 Pose`

**骨骼后缀**：选中骨骼 → 加后缀 / 还原。

**预设系统**：保存当前骨骼选中集合为 JSON 文件 (`bone_presets/` 文件夹)，下次一键选中。

| 按钮 | 说明 |
|---|---|
| **保存** | 弹窗输入预设名，保存选中骨骼 |
| `[🔘]` | 选中该预设记录的所有骨骼（带 `_dm` 后缀也能匹配） |
| `[✕]` | 删除预设 |

---

### 骨骼名称检测 `🟡 Edit`

| 按钮 | 说明 |
|---|---|
| **选中非英文字符骨骼** | 自动选中名称含中文/日文等非 ASCII 字符的骨骼 |
| **标点符号替换为 _** | 空格、标点、特殊符号全部替换为 `_` |

---

### Noesis 骨骼缩放转 RE Mesh `🟢 Object`

独立框。一键执行约束链：选中骨架 → Copy Scale Step1 → Step1 Copy Scale Step2 → 烘焙 Step2 → Step1 缩放 0.01 → 烘焙 Step1。

前提：场景中需有 `Step1` 和 `Step2` 两个骨架。

---

### 部件分离 `🟢 Object`

选中共用同一骨架的网格 → 复制骨架 → 把选中的网格挂到新骨架上 → 父子关系同步迁移。旧骨架保留不动。

---

### 材质工具 `🟢 Object`

| 按钮 | 说明 |
|---|---|
| **材质独立并改名** | 选中网格 → 材质变成单用户副本 → 改名为 `物体名_材质名` |
| **材质独立并合并** | 同上 + 合并为一个物体 |
| **按材质分离** | 按材质槽拆分 → 每个碎片改为材质名 |

---

### 清空与复制 `🟢 Object`

| 按钮 | 说明 |
|---|---|
| **清空模型** | 删除 UV/顶点组/颜色属性/材质/形态键/所有几何体 → 空模型 |
| **复制** | 按输入数量批量复制 |

---

### 快捷工具 `🟢 Object`

**去 .001**：选中物体 → 去掉 Blender 自动加的 `.001` `.002` 后缀（名字冲突时不动）。

---
